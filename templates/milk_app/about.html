This app was made based on the following technologies:

HTML 5

Even bootstrap has to be based on something. And this something is html 5 in this case. Nothing too old, nothing too new. Everyone always knows what to expect from the good old HTML.

Bootstrap 4.2

Bootstrap is responsible for all the responsiveness and adjusting of the web application you can see. Almost everything was done in bootstrap, including image resizing, scrolling functions, nav-bar, etc.

CSS 3

All the styling was made in the css template that is attached to all the .html files. One style to rule them all, one style to find them, one style to bring them all, and in the darkness bind them. *what's up reference*

Django

The whole back-end was built on a python extension - django.
